# dotfiles

Fleshing out my own dotfiles.

## High-Level Objectives

- Make it easy and simple to update dotfiles locally and have those changes be reflected here.
  - The easier this is to maintain, the more worthwhile this project becomes!
- Make it easy and simple to pull these dotfiles down on a new machine. This applies to users who are not me.
  - The longer term goal is to have configurations setup for MacOS, Linux, and possibly even Windows.
